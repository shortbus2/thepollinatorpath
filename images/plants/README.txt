PLANT PHOTOS ARE OPTIONAL

The site works even when this folder is empty. Each plant displays a clean
"Photo coming soon" illustration until its photo is added.

To add a photo, name it with the plant number and use JPG format:
  8.jpg   = Blanket Flower
  10.jpg  = Harebell
  15.jpg  = Creeping Oregon Grape

Put the file directly inside this folder, commit it in GitHub Desktop, and push.
The placeholder will automatically be replaced by the photo.

For the most reliable results:
- use lowercase .jpg
- crop landscape photos approximately 4:3 when possible
- do not change the number in data.js unless the matching filename changes too
