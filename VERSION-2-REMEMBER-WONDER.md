# The Pollinator Path — Version 2: Remember Wonder

This build focuses on polish, dependable publishing, and privacy by design.

## Included

- Garden Walk language and streamlined mobile photo selection
- progress updates and conservative wildlife confidence labels
- clickable Garden Brain dashboard cards and a dedicated task center
- editable, GitHub-synced garden residents including Beth and Little Head Todd
- slow editorial homepage carousel and weekly garden recap foundation
- private-first entry workflow with public publishing disabled by default
- required public-photo privacy review
- separate Safe for Homepage approval
- fresh canvas-generated JPEG copies that discard EXIF/GPS metadata
- randomized public photo filenames
- Worker enforcement preventing private entries from entering the public repository

## Security boundary

The GitHub Pages repository is public. Private drafts and private notes therefore remain on the current device in this release. They are not a durable cross-device private archive. See `SECURITY-PRIVACY.md` before deployment.

## Identification assistance

This release provides cautious suggestions from the selected visitor and prior garden records. True visual classification and automatic visual privacy scanning require a configured vision service on the secure Worker. The interface does not pretend those scans occurred.
