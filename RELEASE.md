# Release 3.2.0 — Ecological Storytelling

**Release date:** 2026-07-15  
**Status:** Candidate build for local and iPhone/Safari quality-control testing

## Release purpose

This release establishes the project’s current design philosophy and its documented software-development workflow.

- **Garden Brain is the private memory.**
- **The Pollinator Path is the curated public story.**
- **Keep the joy:** field capture should feel like noticing the garden, not clerical work.
- **Keep the discipline:** releases, decisions, known issues, and future work must live in the repository rather than only in conversation.
- Public transformation storytelling emphasizes **ecological change**, not a step-by-step landscaping renovation.
