# Security and Privacy

- Public publishing is off by default.
- Browser-generated JPEG copies remove embedded EXIF/GPS metadata.
- Visual details still require human review.
- Homepage eligibility requires separate wide-audience approval.
- Private notes never enter the public GitHub payload.
- Secrets belong in Cloudflare Worker secrets, never source files.
- Do not publish routines, travel plans, access details, exact nest locations, house numbers, plates, mail, or revealing reflections.
- Automatic object detection and blurring are not active unless a real vision service is connected and clearly disclosed in Garden Brain.
